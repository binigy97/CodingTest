## 144 : 반복제어문3 - 형성평가5
### 문제
```
자연수 n을 입력받아 "출력 예"와 같이 n줄에 걸처 오른쪽으로 정렬된 삼각형이 출력되는 프로그램을 작성하시오. 
주의! '*'과 '*'사이에 공백이 없고 줄사이에도 빈줄이 없다.
```
### 입력 예
```
3
```
### 출력 예
```
    *
  ***
*****
```

---

### 풀이1
```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {    
     Scanner sc = new Scanner(System.in);
     int n = sc.nextInt();
     int temp = n * 2 - 1;
     
     for(int i = 1; i <= temp; i += 2) {
    	 System.out.println(" ".repeat(temp - i) + "*".repeat(i));
     }
     sc.close();
 }
}
```
소요시간 : 180ms

### 풀이2
```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {    
     Scanner sc = new Scanner(System.in);
     int n = sc.nextInt();
     int temp = n * 2 - 1;
     
     for(int i = 1; i <= temp; i += 2) {
    	 for(int j = temp - i; j > 0; j--) {
    		 System.out.print(" ");
    	 }
    	 for(int j = temp - i; j < temp; j++) {
    		 System.out.print("*");
    	 }
    	 System.out.println();
     }
     sc.close();
 }
}
```
소요시간 : 269ms

[정올 552번](https://github.com/binigy97/CodingTest/blob/main/Jungol/loop3/552.md)과 비슷한 결과다. 
다른 JAVA 사용자의 실행시간을 확인해 본 결과, 다음과 같다.
- 2022.01 ~ 2022.02 : 400ms 이상, 최대 611ms
- 2022.03 ~ 2022.05 : 270 ~ 320ms
- 풀이1 : 180ms 

풀이1과 50ms 미만으로 차이 나는 결과는 없었다. 
생각보다 차이가 크다. 풀이1을 풀이2로 코드를 수정하여 다른 사람과의 차이를 알아 보았다. 
실행 시간은 0.1s 가량 늘어났다. 

JAVA는 python과 같이 string에 * 연산자가 정의되어져 있지 않아 매우 불편하다. 
**repeat 함수**를 이용한다면 python처럼 간결한 코드를 통해 빠른 결과를 볼 수 있다. **(JDK11 이상부터 사용가능)**
