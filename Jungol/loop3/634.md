## 634 : 반복제어문3 - 자가진단2
### 문제
```
자연수 n을 입력받아서 n줄만큼 다음과 같이 출력하는 프로그램을 작성하시오.
```
### 입력 예
```
5
```
### 출력 예
```
*
**
***
****
*****
```

---

### 풀이1
```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {    
	 Scanner sc = new Scanner(System.in);
	 int n = sc.nextInt();
	 
	 for(int i = 1; i <= n; i++) {
		 System.out.println("*".repeat(i));
	 }
	 sc.close();
 }
}
```
소요시간 : 168ms
### 풀이2
```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {    
	 Scanner sc = new Scanner(System.in);
	 int n = sc.nextInt();
	 
	 for(int i = 1; i <= n; i++) {
		 for(int j = 1; j <= i; j++) {
			 System.out.print("*");
		 }
		 System.out.println();
	 }
	 sc.close();
 }
}
```
소요시간 : 162ms

#### :question: 단일 반복문에 repeat() 함수 vs 이중 반복문 : 연산 속도 차이는?
두 경우의 실행 속도 차이가 궁금했다. 함수 호출보다 for문의 동작이 더 빠를 것이라고 생각했지만, **유의미한 차이는 없었다.** 반복 실행해도 의미를 못 찾을 것 같아서 별도의 테스트는 해보지 않았다.  
위와 같은 경우에 repeat() 함수를 사용하는 게 실행 속도, 코드의 간결성 측면에서 이득이 될 것 같다.
