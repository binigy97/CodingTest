## 143 : 반복제어문3 - 형성평가4
### 문제
```
자연수 n을 입력받아 "출력 예"와 같이 출력되는 프로그램을 작성하시오.

주의! '*'과 '*'사이에 공백이 없고 줄사이에도 빈줄이 없다.
```
### 입력 예
```
3
```
### 출력 예
```
*****
 ***
  *
 ***
*****
```

---

### 풀이
```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {    
     Scanner sc = new Scanner(System.in);
     int n = sc.nextInt();
     int tmp = n * 2 - 1;
     
     for(int i = 0; i < n; i++) {
    	 System.out.println(" ".repeat(i) + "*".repeat(tmp) + " ".repeat(i));
    	 tmp -= 2;
     }
     tmp = 3;
     for(int i = n - 1; i > 0; i--) {
    	 System.out.println(" ".repeat(i - 1) + "*".repeat(tmp) + " ".repeat(i - 1));
    	 tmp += 2;
     }
     sc.close();
 }
}
```

**repeat 함수**를 이용한다면 python처럼 간결한 코드를 통해 빠른 결과를 볼 수 있다. **(JDK11 이상부터 사용가능)**
