## 1102 : 스택 (stack)
### 문제
```
(생략)

<처리조건>

주어진 명령은 다음의 3가지이다.

1. "i a"는 a라는 수를 스택에 넣는다. 이때, a는 10,000 이하의 자연수이다.

2. "o"는 스택에서 데이터를 빼고, 그 데이터를 출력한다. 만약 스택이 비어있으면, "empty"를 출력한다.

3. "c"는 스택에 쌓여있는 데이터의 수를 출력한다.
```
### 입력형식
```
첫 줄에 N이 주어진다. N은 주어지는 명령의 수이다. (1≤N≤100)

둘째 줄부터 N+1줄까지 N개의 명령이 주어지는데, 한 줄에 하나씩 주어진다.
```
### 출력형식
```
각 명령에 대한 출력 값을 한 줄에 하나씩 출력한다.

출력내용이 하나도 없는 경우는 주어지지 않는다.
```
### 입력 예
```
7
i 7
i 5
c
o
o
o
c
```
### 출력 예
```
2
5
7
empty
0
```
---

### 풀이1
```java
import java.util.Scanner;

public class Main {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		Stack st = new Stack();

		int cnt = sc.nextInt();
		sc.nextLine();
		for (int i = 0; i < cnt; i++) {
			String flag = sc.nextLine(); // 명령어

			switch (flag.charAt(0)) {
			// input item
			case 'i':
				// 입력 명령어의 숫자를 10진수 int로 형변환 하여 배열에 삽입
				String[] tmp = flag.split(" ");
				int n = Integer.parseInt(tmp[1]);
				st.push(n);
				break;
			// out item
			case 'o':
				System.out.println(st.top());
				st.pop();
				break;
			// count item
			case 'c':
				System.out.println(st.size());
			}
		}
		sc.close();
	}
}

class Stack {
	private int[] st; // stack memory
	private int top; // input location of item

	public Stack() {
		this.st = new int[100];
		this.top = 0;
	}

	// 스택이 비었는지 확인
	public boolean empty() {
		if (this.top <= 0) {
			return true;
		}
		return false;
	}

	// 스택에 담긴 원소의 개수 반환
	public int size() {
		if (empty()) {
			return 0;
		}
		return this.top;
	}

	// 최근 스택에 담긴 원소 반환
	public String top() {
		if (empty()) {
			return "empty";
		}
		return String.valueOf(this.st[this.top - 1]);
	}

	// 스택에 원소 추가
	public void push(int data) {
		this.st[this.top] = data;
		top++;
	}

	// 최근 스택에 담은 원소 삭제
	public void pop() {
		if (empty()) {
			this.top = 0;
		} else {
			this.top--;
		}
	}
}
```
